Python PluginAPI using ctypes
Totte Karlsson (tottek@gmail.com)

The file rrPlugins.py is a python wrapper for RoadRunners pluginlib_c_api library.

==== IMPORTANT ====
To use this script in your own scripts, or if you are to run the example scripts, 
make the folder in where it is located is added to your PYTHONPATH, 
	e.g. add c:\roadrunner1.0.0\plugins\python to your PYTHONPATH
=================================================================================

The Example folder contain Examples illustrating how to get access and use the pluginlib and plugins.

See further API documentation on the web, http://libroadrunner.org/ on how to use this wrapper.
